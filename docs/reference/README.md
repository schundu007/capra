## System Requirements

### Web Application

| Component | Minimum | Recommended |
|-----------|---------|-------------|
| Browser | Chrome 90+, Firefox 88+, Safari 14+ | Latest version |
| Internet | 1 Mbps | 5+ Mbps |
| Screen | 1280x720 | 1920x1080+ |

### Desktop Application

**macOS:**
| Component | Minimum | Recommended |
|-----------|---------|-------------|
| OS | macOS 11 (Big Sur) | macOS 13+ (Ventura) |
| RAM | 4 GB | 8+ GB |
| Disk | 200 MB | 500 MB |
| CPU | Apple Silicon or Intel | Apple Silicon |

**Windows:**
| Component | Minimum | Recommended |
|-----------|---------|-------------|
| OS | Windows 10 (64-bit) | Windows 11 |
| RAM | 4 GB | 8+ GB |
| Disk | 200 MB | 500 MB |
| CPU | 2 GHz dual-core | 4+ cores |

## Supported Platforms

### Problem Sources

| Platform | Support | Features |
|----------|---------|----------|
| LeetCode | Full | URL extraction, all problem types |
| HackerRank | Full | URL extraction, contests |
| CodeSignal | Partial | Manual paste only |
| Other | Manual | Copy/paste problem text |

### Video Call Integration

| Platform | Compatibility | Notes |
|----------|---------------|-------|
| Zoom | Full | All features supported |
| Google Meet | Full | Works with Interview Assistant |
| Microsoft Teams | Full | All features supported |
| WebEx | Partial | Some audio issues possible |
