# Keyboard Shortcuts

Complete reference of all keyboard shortcuts in Ascend.

---

## Global Shortcuts

These shortcuts work from anywhere in the application.

### Mode Switching

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + 1` | Switch to Coding mode |
| `Cmd/Ctrl + 2` | Switch to System Design mode |
| `Cmd/Ctrl + 3` | Switch to Preparation mode |

### Navigation

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Focus problem input |
| `Cmd/Ctrl + ,` | Open settings |
| `Cmd/Ctrl + /` | Open keyboard shortcuts reference |
| `Escape` | Close modal / Cancel action |

### Actions

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + Enter` | Solve / Generate |
| `Cmd/Ctrl + Shift + S` | Take screenshot (Desktop) |
| `Cmd/Ctrl + Shift + C` | Copy solution code |

---

## Coding Mode Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + Enter` | Generate solution |
| `Cmd/Ctrl + R` | Run code |
| `Cmd/Ctrl + Shift + R` | Run with custom input |
| `Tab` | Switch between Code/Explanation tabs |

---

## System Design Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + Enter` | Generate design |
| `Cmd/Ctrl + D` | Download diagram |
| `Cmd/Ctrl + Shift + D` | Regenerate diagram |

---

## Interview Assistant Shortcuts

> **Note:** Desktop app only (Elite plan)

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + I` | Toggle Interview Assistant |
| `Cmd/Ctrl + Shift + L` | Start/Stop listening |
| `Cmd/Ctrl + Shift + H` | Toggle stealth mode |
| `Cmd/Ctrl + Shift + ↑` | Increase overlay opacity |
| `Cmd/Ctrl + Shift + ↓` | Decrease overlay opacity |
| `Cmd/Ctrl + Shift + M` | Move overlay position |

---

## Text Editing

Standard text editing shortcuts work in all input fields:

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + A` | Select all |
| `Cmd/Ctrl + C` | Copy |
| `Cmd/Ctrl + V` | Paste |
| `Cmd/Ctrl + X` | Cut |
| `Cmd/Ctrl + Z` | Undo |
| `Cmd/Ctrl + Shift + Z` | Redo |

---

## Desktop App Only

These shortcuts are exclusive to the desktop application:

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + Q` | Quit application |
| `Cmd/Ctrl + W` | Close window |
| `Cmd/Ctrl + M` | Minimize window |
| `Cmd/Ctrl + Shift + F` | Toggle fullscreen |
| `Cmd/Ctrl + +` | Zoom in |
| `Cmd/Ctrl + -` | Zoom out |
| `Cmd/Ctrl + 0` | Reset zoom |

---

## Platform-Specific Keys

| Symbol | macOS | Windows |
|--------|-------|---------|
| `Cmd` | Command (⌘) | — |
| `Ctrl` | Control (⌃) | Ctrl |
| `Alt` | Option (⌥) | Alt |
| `Shift` | Shift (⇧) | Shift |

---

## Customizing Shortcuts

Currently, keyboard shortcuts cannot be customized. Custom shortcut support is planned for a future release.

To request a shortcut change or addition, please [open an issue](https://github.com/schundu007/capra/issues).
